# Pocket Count

ブラウザだけで動く、用途別カウンターです。通常カウント、スト6、スマブラの対戦結果を端末内に保存できます。

## ファイル構成

- `index.html` — 公開ページの入口と最小限のHTML
- `assets/styles.css` — UI、レスポンシブ、セーフエリア
- `src/data.js` — プリセット、スマブラのファイター一覧、50音グループ
- `src/core.js` — 状態、保存、データ移行、集計、描画
- `src/app.js` — イベント接続、起動、別タブ同期
- `tests/smoke.html` — ブラウザで開く簡易スモークテスト
- `app.html` — 旧URLから `index.html` への互換転送

## 開発方針

ビルド工程や外部依存を持たない静的構成です。ファイルを直接編集し、静的サーバーで確認できます。保存キーは既存版から変更していないため、分割前のローカルデータを引き継ぎます。
